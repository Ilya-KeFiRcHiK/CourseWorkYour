Методы строк

toUpperCase slice toLowerCase charAt trim split concat substring repeat replace join

Метод toUpperCase
Преобразование строки в верхний регистр

let str = 'abcde';
console.log(str.toUpperCase());
Результат выполнения кода:
'ABCDE'

Метод slice

Вырезание и возвращение указанной части массива. Сам массив при этом не изменяется.

let str = 'abcde';
let res = str.slice(0, 1).toUpperCase() + str.slice(1);
console.log(res);
Результат выполнения кода:
'Abcde'

Метод toLowerCase

Преобразование строки в нижний регистр

let str = 'ABCDE';
console.log(str.toLowerCase());
Результат выполнения кода:
'abcde'

Метод charAt

Возвращение символа в строке

let str = 'abcde';
let char = str.charAt(0);
console.log(char);
Результат выполнения кода:
'a'

Метод trim()

Удаление пробелов по краям строки. Чаще всего это нужно при вводе пользователем каких-либо значений: он может случайно налепить

лишних пробелов - и наша задача очистить введенный текст от них.

Удалим концевые пробелы у строки:

let str = ' abcde ';
console.log(str.trim());
Результат выполнения кода (пробелы пропали):
'abcde'

Метод split()

Разбивание строки в массив по указанному в первом параметре разделителю. Если он не задан - вернется вся строка. Если он задан как

пустые кавычки, то каждый символ строки попадет в отдельный элемент массива. Вторым необязательным параметром можно указать

максимальное количество элементов в получившемся массиве.

Пусть дана некоторая строка с дефисами. Разобьём эту строку в массив по разделителю '-':

let str = 'ab-cd-ef';
let arr = str.split('-');
Результат выполнения кода:
['ab', 'cd', 'ef']
console.log(arr);

Метод concat

Соединение указанные строки. Количество строк для объединения не ограничено. По сути действие этого метода аналогично операции '+'

для слияния строк.

Давайте с помощью метода concat сольем три строки в одну:

let str1 = 'a';
let str2 = 'b';
let str3 = 'c';
let res = str1.concat(str2, str3);
console.log(res);
Результат выполнения кода:
'abc'

Метод substring

Возвращение подстроки из строки (исходная строка при этом не изменяется). Первый параметр задает номер символа, с которого метод

начинает отрезать (нумерация идет с нуля), а второй параметр - номер символа, на котором следует закончить вырезание (символ с этим

номером не включается в вырезанную часть). Второй параметр не является обязательным, если он не указан, то вырезаны будут все

символы до конца строки.

Dырежем из нее символы с первого по третий:

let str = 'abcde';
let sub = str.substring(1, 3);
console.log(sub);
Результат выполнения кода (символ с номером 3 не включится в вырезанную часть):
'bc'

Метод repeat

Создание новой строки, содержащую указанное количество копий первоначальной строки, слитых вместе.

Укажем в параметре метода число 1:

let res = 'abcde'.repeat(1);
console.log(res);
В результате выполнения кода наша строка останется прежней:
'abcde'

Теперь мы скопируем строку 2 раза:

let res = 'abcde'.repeat(2);
console.log(res);
Результат выполнения кода:
'abcdeabcde'

Метод replace

Осуществление поиска и замены частей строки. Первым параметром принимается регулярное выражение, а вторым - подстрока, на

которую заменяем.

Найдем и заменим символ 'a':

let str = 'bab';
let res = str.replace(/a/, '!');
console.log(res);
Результат выполнения кода:
'b!b'

По умолчанию заменяется только первое совпадение. Давайте снова заменим символ 'a' в строке:

let str = 'baaab';

let res = str.replace(/a/, '!');

console.log(res);

Результат выполнения кода:

'b!aab'

Метод join

Объединение элементов массива в строку с указанным разделителем (он будет вставлен между элементами массива). Разделитель

задается параметром метода и не является обязательным. Если он не задан - по умолчанию в качестве разделителя возьмется запятая.

Если вы хотите слить элементы массива без разделителя - укажите его как пустую строку ''.

Пусть дан некоторый массив. Объединим элементы этого массива в строку с разделителем '-':

let arr = [1, 2, 3];
let str = arr.join('-');
console.log(str);
Результат выполнения кода:
'1-2-3'

Не укажем разделитель и по умолчанию разделителем станет запятая:

let arr = [1, 2, 3];
let str = arr.join();
console.log(str);
Результат выполнения кода:
'1,2,3'

8-800-700-93-29
