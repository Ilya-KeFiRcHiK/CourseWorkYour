Переменные


Основным понятием любого языка программирования является переменная.
Переменная - это контейнер, в котором мы можем хранить какие-либо данные, например, строки или числа.
Каждая переменная должна иметь имя, которое может состоять из латинских букв, чисел, символов $ и знаков подчеркивания. При этом первый символ имени переменной не должен быть цифрой.


str my str myStr a1 $ $a $$a

Для того, чтобы использовать переменную необходимо:
- объявить: написать перед ее именем ключевое слово.

let А
- присвоить ей значение:
А = 5
- вывести значение переменной на экран:

alert(А); let А; // объявим переменную

А = 3; // присвоим ей значение

аlert (А); // выведем значение переменной на экран

Можно объявить несколько переменных:
let А = 5;

let В = 10;

let С = 15;

let А, В, С; // объявляем все 3 переменные

Можно изменить значение переменной:
let А;// объявляем переменную

А = 2;// записываем в нее значение 2

alert(А);// выведет 2

А = 5;// записываем теперь значение 5, затирая значение 2

alert(А); // выведет 5

let А = 1;
